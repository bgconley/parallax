# Phase 9 Dynamic App UAT Matrix

This matrix is the canonical checklist for the Phase 9 corrective UAT pass that proves Parallax is a dynamic temporal application, not a hardcoded Figma/example playback.

## Scope

Run against the iOS simulator connected to the GPU-node backend through `http://127.0.0.1:18000`.

Required proof for mutating flows:

- Simulator screenshot or UI hierarchy snapshot for the user-visible state.
- Backend row proof from the GPU-node Postgres container.
- Canonical source event, annotation, decision, answer, or workflow row where applicable.
- No runtime fixture leakage from Figma/example scenarios such as wash/kitchen/sponge unless the UAT user entered that exact text.
- `client_device_id` matches the UAT device for all source events created during the scenario.
- Mutation replay/idempotency is preserved for sync/retry scenarios.

Required presentation proof across all simulator paths:

- User-facing labels must be display copy, not raw enum, variable, or contract identifiers such as `possible_forgotten_timer`, `needs_attention`, or snake_case workflow names.
- Button text must fit without truncation, and button label size must be proportional to button height, adjacent controls, and expected tap affordance.
- Adjacent buttons must use balanced sizing and spacing; no primary/secondary pair should look accidentally disproportionate because of wording length.
- Text, badges, rows, and drawer controls must use enough whitespace to avoid cramped borders, overlap, clipping, or accidental visual grouping.
- Dynamic runtime content must be inspected in screenshots for alignment, text fit, and readability, not only for successful taps or backend mutations.

## Status Legend

| Status | Meaning |
| --- | --- |
| `PENDING` | Not yet executed in this matrix pass. |
| `PASS` | Simulator behavior and backend proof matched expected behavior. |
| `FAIL` | Behavior is wrong and requires remediation. |
| `BLOCKED` | Could not execute because prerequisite state/tooling was unavailable. |
| `PARTIAL` | Some proof exists but coverage is incomplete. |
| `INTENTIONAL` | UI element is intentionally display-only or disabled, with no mutation expected. |
