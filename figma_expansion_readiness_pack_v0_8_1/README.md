# Figma Expansion Readiness Pack v0.8.1

This pack is the corrected, simplified, verified handoff for continuing the app in Figma without design drift.

Use this pack **in addition to** the previously approved mockups, but not as one flat pile. Treat the materials as a ranked system:

1. **Anchors** — the approved canonical screens. These define the emotional and visual truth of the app.
2. **Source docs** — the textual truth for design language, interaction rules, accessibility, and acceptance gates.
3. **System boards** — visual grammar for components, states, and layout patterns.
4. **Continuation screens** — safe sibling examples for extrapolating adjacent surfaces.
5. **Tokens / inventory / contracts** — exact implementation rails for Figma and later code.

This pack exists because prior nested file links were unreliable. Everything here is repackaged into one coherent, navigable folder with simple filenames.

Start with `START_HERE_FIGMA_AGENT.md`.
